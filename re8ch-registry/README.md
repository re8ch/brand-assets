# Re8ch Registry brand asset

Static and animated marks for the Re8ch Registry product surface.

- `svg/re8ch-registry-mark.svg`: animated SVG mark for product pages, favicons, and product cards.
- `motion/re8ch-registry-motion.css`: optional host-page animation helpers when the SVG is embedded inline.

## Design Rules

- Match the existing RE8CH / Ledger asset language: transparent background,
  black structure edge, compact geometry, and a small set of brand color fills.
- Keep the icon legible at 24 px and 48 px.
- Use one product metaphor only: a harbor watchtower guarding signed container flow.
- Motion is subtle: lighthouse beam opacity plus a small signal-light pulse.

The mark references Harbor-style registry ideas without copying Harbor: a port
watchtower, watched entrance, and signal beam for OCI flow. It intentionally
avoids node maps, orbits, shadows, filters, and dense routing diagrams.
